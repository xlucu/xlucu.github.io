## 1️⃣ Vercel (الأسهل - موصى به) ⭐⭐⭐⭐⭐

### من الموقع:
1. اذهب إلى: **https://vercel.com**
2. سجل دخول بحساب GitHub
3. اضغط "Add New" → "Project"
4. اختر المشروع
5. اضغط "Deploy"
6. ✅ **تم!**

### من Terminal:
```bash
npm install -g vercel
vercel login
vercel --prod
```

**الرابط:** `https://your-project.vercel.app`

---

## 2️⃣ Netlify (سهل جداً)

### الطريقة السريعة (Drag & Drop):
```bash
npm run build
```
ثم اذهب إلى: **https://app.netlify.com/drop**
واسحب مجلد `dist`

### من GitHub:
1. اذهب إلى **https://netlify.com**
2. "Add new site" → "Import from Git"
3. اختر المشروع
4. اضغط "Deploy"
