# أكاديمية المواهب - موقع أكاديمية كرة قدم احترافي

موقع ويب تسويقي كامل لأكاديمية كرة قدم تستهدف جميع الأعمار، يوفر معلومات عن البرامج التدريبية والتسجيل المباشر عبر واتساب.

**Experience Qualities**: 
1. **طاقة رياضية** - تصميم ديناميكي يعكس حماس وشغف كرة القدم
2. **احترافي وواضح** - معلومات منظمة وسهلة الوصول للأهالي واللاعبين
3. **سريع ومباشر** - تجربة سلسة تؤدي مباشرة للتسجيل عبر واتساب

**Complexity Level**: Content Showcase (information-focused)
موقع تسويقي يعرض خدمات الأكاديمية مع دعوات واضحة للإجراء (CTA) للتسجيل عبر واتساب.

## Essential Features

**Hero Section - القسم الرئيسي**
- Functionality: عرض رسالة الأكاديمية الرئيسية مع دعوة فورية للتسجيل وعرض إحصائيات الطلاب والمدربين
- Purpose: جذب انتباه الزوار وتحفيزهم للتسجيل وبناء المصداقية من خلال عرض عدد الطلاب المسجلين والمدربين
- Trigger: عند فتح الموقع مباشرة
- Progression: رؤية Hero → قراءة العنوان والوصف → مشاهدة العداد المتحرك للطلاب والمدربين → النقر على زر "سجل الآن" → فتح واتساب
- Success criteria: زر واتساب يعمل ويفتح محادثة مع الرقم المحدد، عداد الأرقام يتحرك من 0 إلى الرقم المستهدف بطريقة سلسة

**Training Categories - الفئات التدريبية**
- Functionality: عرض 3 فئات عمرية مع التفاصيل والأسعار
- Purpose: مساعدة الأهالي على اختيار الفئة المناسبة لأطفالهم
- Trigger: التمرير للأسفل من Hero
- Progression: رؤية الكروت → مقارنة الفئات → اختيار فئة → النقر على "احجز تجربة مجانية" → فتح واتساب
- Success criteria: 3 كروت واضحة، كل زر يفتح واتساب مع رسالة مخصصة للفئة

**What We Offer - ماذا نقدم**
- Functionality: عرض مميزات وخدمات الأكاديمية
- Purpose: بناء الثقة وإظهار القيمة المقدمة
- Trigger: التمرير بعد قسم الفئات
- Progression: رؤية الأيقونات والنصوص → فهم الخدمات المقدمة
- Success criteria: 4 عناصر بأيقونات واضحة ونصوص مختصرة

**Gallery - معرض الصور**
- Functionality: عرض معرض صور للأكاديمية والملاعب والتدريبات مع إمكانية التصفية
- Purpose: إعطاء الزوار صورة حقيقية عن المرافق والأجواء التدريبية
- Trigger: التمرير بعد قسم "ماذا نقدم" أو النقر على "المعرض" في القائمة
- Progression: رؤية شبكة الصور → اختيار فلتر (الكل/الملاعب/التدريبات/المرافق) → النقر على صورة → فتح عارض الصور بحجم كامل → التنقل بين الصور → إغلاق العارض
- Success criteria: 9 صور بجودة عالية، فلترة تعمل بسلاسة، عارض صور احترافي مع أزرار تنقل

**Our Stars - نجومنا**
- Functionality: عرض قصص نجاح لاعبين سابقين
- Purpose: بناء مصداقية وإلهام اللاعبين الجدد
- Trigger: التمرير بعد قسم "ماذا نقدم"
- Progression: رؤية صور اللاعبين → قراءة قصص النجاح
- Success criteria: 3 كروت لاعبين بصور ونصوص جذابة

**Contact Section - تواصل معنا**
- Functionality: عرض معلومات الاتصال + خريطة + فورم تسجيل
- Purpose: تسهيل التواصل والتسجيل
- Trigger: التمرير للنهاية أو النقر على "تواصل معنا"
- Progression: رؤية المعلومات → ملء الفورم أو النقر على واتساب → إرسال البيانات
- Success criteria: خريطة Google Maps تعمل، فورم يفتح واتساب مع البيانات

**Footer - التذييل**
- Functionality: روابط مواقع التواصل الاجتماعي
- Purpose: زيادة التواصل عبر المنصات الاجتماعية
- Trigger: في أسفل كل صفحة
- Progression: رؤية الأيقونات → النقر على رابط → فتح صفحة التواصل
- Success criteria: 3 أيقونات واضحة لـ Facebook, Instagram, TikTok

## Edge Case Handling
- **رقم واتساب غير صحيح**: التحقق من الرقم الدولي وإضافة رمز البلد تلقائياً
- **الفورم فارغ**: عدم فتح واتساب إلا بعد ملء جميع الحقول
- **الخريطة لا تعمل**: عرض العنوان النصي كبديل
- **الجوال الصغير**: تصميم responsive يعيد ترتيب العناصر بشكل عمودي
- **بطء الاتصال**: استخدام صور محسّنة وخلفيات CSS بدلاً من الصور الثقيلة

## Design Direction
تصميم عصري وأنيق يجمع بين الاحترافية والرقي، مع لمسات حديثة تعكس التميز والطموح.

## Color Selection
تصميم عصري وأنيق بألوان بنفسجية وزهرية راقية تعكس الاحترافية والتميز.

- **Primary Color**: بنفسجي غني `oklch(0.55 0.18 260)` - يمثل الطموح والاحترافية والتميز
- **Secondary Color**: بنفسجي فاتح `oklch(0.72 0.16 285)` - يضيف عمق وتدرج أنيق
- **Accent Color**: زهري ديناميكي `oklch(0.68 0.20 320)` - للتركيز والحيوية، يجذب الانتباه
- **Background**: خلفية فاتحة `oklch(0.98 0.005 280)` - نظيفة ومريحة للعين
- **Foreground/Background Pairings**: 
  - Primary (بنفسجي): White text (#FFFFFF) - Ratio 6.1:1 ✓
  - Accent (زهري): White text (#FFFFFF) - Ratio 5.3:1 ✓
  - Background (فاتح): Dark text - Ratio 15.2:1 ✓

## Font Selection
خط عربي عصري وأنيق يعكس الحداثة والاحترافية مع سهولة القراءة.

- **Typographic Hierarchy**:
  - H1 (العنوان الرئيسي): Tajawal Black/72px/tight، للعنوان البطولي والمميز
  - H2 (عناوين الأقسام): Tajawal Black/56px/normal، لعناوين الأقسام الرئيسية
  - H3 (عناوين الكروت): Tajawal Bold/24px/normal، لعناوين الفئات والكروت
  - Body (النص العادي): Tajawal Medium/18px/relaxed، للوصف والمحتوى
  - CTA Buttons (أزرار الإجراء): Tajawal Black/20px/normal، للتأكيد على الأكشن

## Animations
حركات سلسة وراقية تعكس الاحترافية والحداثة، مع تركيز على transitions ناعمة وتأثيرات بصرية جذابة.

- **Hover على الأزرار**: تكبير سلس مع ظل ناعم (scale 1.1) لإحساس بالتفاعل الراقي
- **Scroll Reveals**: ظهور العناصر بطريقة سلسة مع fade-in وslide-in عند التمرير
- **Hero Section**: خلفية متدرجة مع overlays متحركة ناعمة وتأثيرات blur
- **Cards Hover**: رفع خفيف مع ظل ثلاثي الأبعاد وتدوير طفيف للأيقونات
- **Gradient Animations**: تأثيرات gradients متحركة وناعمة في الخلفيات
- **Glass Effects**: تأثيرات زجاجية شفافة (glassmorphism) للعناصر التفاعلية

## Component Selection
- **Components**: 
  - Button (shadcn): للأزرار الرئيسية مع تخصيص أخضر
  - Card (shadcn): للفئات التدريبية وكروت اللاعبين
  - Input + Label + Textarea (shadcn): للفورم
  - Separator (shadcn): للفواصل بين الأقسام
  
- **Customizations**:
  - خلفية gradient للـ Hero مع صورة overlay
  - Grid pattern بأسلوب خطوط الملعب في الخلفية
  - أيقونات من Phosphor Icons بأسلوب رياضي
  
- **States**:
  - Default: أزرار خضراء صلبة مع نص أبيض
  - Hover: تكبير + ظل أقوى
  - Active: تقليص خفيف للضغط
  - Focus: حلقة خضراء مضيئة
  
- **Icon Selection**:
  - Soccer (كرة قدم) للأيقونة الرئيسية
  - Users (مجموعة) للفئات العمرية
  - Trophy (كأس) للنجاح
  - WhatsApp logo للتواصل
  - MapPin للموقع
  
- **Spacing**: 
  - Sections: py-20 للمساحات الكبيرة
  - Cards: gap-8 بين الكروت
  - Content: gap-4 للمحتوى الداخلي
  - Buttons: px-8 py-4 للأزرار الكبيرة
  
- **Mobile**: 
  - Hero: عمودي بالكامل مع نص مركز
  - Grid: من 3 أعمدة إلى عمود واحد
  - Text: تصغير الخطوط 25%
  - Spacing: تقليل الـ padding للنصف
