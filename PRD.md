# أكاديمية المواهب - موقع أكاديمية كرة قدم احترافي

موقع ويب تسويقي كامل لأكاديمية كرة قدم تستهدف جميع الأعمار، يوفر معلومات عن البرامج التدريبية والتسجيل المباشر عبر واتساب.

**Experience Qualities**: 
1. **طاقة رياضية** - تصميم ديناميكي يعكس حماس وشغف كرة القدم
2. **احترافي وواضح** - معلومات منظمة وسهلة الوصول للأهالي واللاعبين
3. **سريع ومباشر** - تجربة سلسة تؤدي مباشرة للتسجيل عبر واتساب

**Complexity Level**: Content Showcase (information-focused)
موقع تسويقي يعرض خدمات الأكاديمية مع دعوات واضحة للإجراء (CTA) للتسجيل عبر واتساب.

## Essential Features

**Hero Section - القسم الرئيسي**
- Functionality: عرض رسالة الأكاديمية الرئيسية مع دعوة فورية للتسجيل
- Purpose: جذب انتباه الزوار وتحفيزهم للتسجيل
- Trigger: عند فتح الموقع مباشرة
- Progression: رؤية Hero → قراءة العنوان والوصف → النقر على زر "سجل الآن" → فتح واتساب
- Success criteria: زر واتساب يعمل ويفتح محادثة مع الرقم المحدد

**Achievements Section - قسم الإنجازات**
- Functionality: عرض إنجازات الأكاديمية بأرقام متحركة (عدد الطلاب، البطولات، المدربين، نسبة الرضا)
- Purpose: بناء المصداقية والثقة من خلال إظهار الإنجازات الملموسة للأكاديمية
- Trigger: التمرير بعد Hero Section
- Progression: رؤية القسم → تفعيل عداد الأرقام المتحرك → مشاهدة الأرقام تتزايد بطريقة سلسة → قراءة وصف كل إنجاز
- Success criteria: الأرقام تتحرك من 0 إلى القيمة المستهدفة بطريقة سلسة عند ظهور القسم، التأثيرات الحركية واضحة وجذابة، البطاقات تظهر بترتيب متتالي

**Training Categories - الفئات التدريبية**
- Functionality: عرض 3 فئات عمرية مع التفاصيل والأسعار
- Purpose: مساعدة الأهالي على اختيار الفئة المناسبة لأطفالهم
- Trigger: التمرير للأسفل من Hero
- Progression: رؤية الكروت → مقارنة الفئات → اختيار فئة → النقر على "احجز تجربة مجانية" → فتح واتساب
- Success criteria: 3 كروت واضحة، كل زر يفتح واتساب مع رسالة مخصصة للفئة

**What We Offer - ماذا نقدم**
- Functionality: عرض مميزات وخدمات الأكاديمية
- Purpose: بناء الثقة وإظهار القيمة المقدمة
- Trigger: التمرير بعد قسم الفئات
- Progression: رؤية الأيقونات والنصوص → فهم الخدمات المقدمة
- Success criteria: 4 عناصر بأيقونات واضحة ونصوص مختصرة

**Gallery - معرض الصور**
- Functionality: عرض معرض صور للأكاديمية والملاعب والتدريبات مع إمكانية التصفية
- Purpose: إعطاء الزوار صورة حقيقية عن المرافق والأجواء التدريبية
- Trigger: التمرير بعد قسم "ماذا نقدم" أو النقر على "المعرض" في القائمة
- Progression: رؤية شبكة الصور → اختيار فلتر (الكل/الملاعب/التدريبات/المرافق) → النقر على صورة → فتح عارض الصور بحجم كامل → التنقل بين الصور → إغلاق العارض
- Success criteria: 9 صور بجودة عالية، فلترة تعمل بسلاسة، عارض صور احترافي مع أزرار تنقل

**Our Stars - نجومنا**
- Functionality: عرض قصص نجاح لاعبين سابقين
- Purpose: بناء مصداقية وإلهام اللاعبين الجدد
- Trigger: التمرير بعد قسم "ماذا نقدم"
- Progression: رؤية صور اللاعبين → قراءة قصص النجاح
- Success criteria: 3 كروت لاعبين بصور ونصوص جذابة

**Testimonials Section - شهادات الأهالي**
- Functionality: عرض تقييمات وشهادات حقيقية من أهالي اللاعبين
- Purpose: بناء الثقة والمصداقية من خلال تجارب واقعية للعائلات
- Trigger: التمرير بعد قسم "نجومنا"
- Progression: رؤية كروت الشهادات → قراءة آراء الأهالي → رؤية التقييمات بالنجوم → التشجيع للانضمام
- Success criteria: 6 شهادات متنوعة، كل شهادة تحتوي على اسم الأهل، صورة رمزية، تقييم 5 نجوم، ونص الشهادة، مع عداد للعائلات السعيدة في الأسفل

**Admin Panel - لوحة تحكم الإدمن**
- Functionality: لوحة تحكم سرية ومحمية بكلمة مرور تمكن الإدمن من تعديل كامل محتوى الموقع
- Purpose: إتاحة التعديل على المحتوى بدون الحاجة للبرمجة أو مساعدة خارجية
- Trigger: الضغط على أزرار لوحة المفاتيح بتسلسل "admin" من أي صفحة في الموقع
- Progression: كتابة "admin" → ظهور نافذة تسجيل الدخول → إدخال البريد وكلمة المرور → الوصول إلى لوحة التحكم → التعديل على الأقسام المختلفة → حفظ التغييرات
- Success criteria: 
  - تسجيل دخول محمي ببريد إلكتروني وكلمة مرور (mohamadyahia209@gmail.com / Mm12345#)
  - واجهة شاملة بـ 8 تبويبات: Hero, الإنجازات, الفئات, المميزات, النجوم, التقييمات, التواصل, السوشيال
  - إمكانية التعديل على جميع النصوص والأرقام والمحتوى
  - إمكانية إضافة وحذف عناصر (فئات، مميزات، نجوم، تقييمات)
  - حفظ تلقائي للبيانات باستخدام Spark KV
  - رسائل تأكيد عند الحفظ أو الحذف
  - مخفية تماماً عن الزوار العاديين

**Contact Section - تواصل معنا**
- Functionality: عرض معلومات الاتصال + خريطة + فورم تسجيل
- Purpose: تسهيل التواصل والتسجيل
- Trigger: التمرير للنهاية أو النقر على "تواصل معنا"
- Progression: رؤية المعلومات → ملء الفورم أو النقر على واتساب → إرسال البيانات
- Success criteria: خريطة Google Maps تعمل، فورم يفتح واتساب مع البيانات

**Footer - التذييل**
- Functionality: روابط مواقع التواصل الاجتماعي
- Purpose: زيادة التواصل عبر المنصات الاجتماعية
- Trigger: في أسفل كل صفحة
- Progression: رؤية الأيقونات → النقر على رابط → فتح صفحة التواصل
- Success criteria: 3 أيقونات واضحة لـ Facebook, Instagram, TikTok

## Edge Case Handling
- **رقم واتساب غير صحيح**: التحقق من الرقم الدولي وإضافة رمز البلد تلقائياً
- **الفورم فارغ**: عدم فتح واتساب إلا بعد ملء جميع الحقول
- **الخريطة لا تعمل**: عرض العنوان النصي كبديل
- **الجوال الصغير**: تصميم responsive يعيد ترتيب العناصر بشكل عمودي
- **بطء الاتصال**: استخدام صور محسّنة وخلفيات CSS بدلاً من الصور الثقيلة

## Performance Optimizations - تحسينات الأداء
لضمان أسرع وقت تحميل وأفضل تجربة مستخدم، تم تطبيق التحسينات التالية:

**Lazy Loading للصور**
- Functionality: تحميل الصور فقط عند اقتراب المستخدم من رؤيتها في الشاشة
- Purpose: تقليل وقت التحميل الأولي وتوفير عرض النطاق الترددي
- Implementation: 
  - مكون LazyImage مخصص يستخدم Intersection Observer API
  - تحميل الصور بـ 50px قبل ظهورها في viewport
  - placeholder مع تأثير animate-pulse أثناء التحميل
  - تأثير fade-in سلس عند اكتمال تحميل الصورة
- Success criteria: الصور تظهر بسلاسة فقط عند الحاجة، الصفحة الأولية تحمل بسرعة أكبر

**Code Splitting بـ React.lazy**
- Functionality: تقسيم كود المكونات وتحميلها عند الطلب
- Purpose: تقليل حجم الحزمة الأولية (initial bundle) وتسريع التحميل الأول
- Implementation:
  - استخدام React.lazy() لتحميل المكونات الكبيرة بشكل منفصل
  - Suspense مع fallback جميل (spinner مع ألوان الموقع)
  - المكونات المقسمة: AchievementsSection, TrainingCategories, WhatWeOffer, Gallery, OurStars, TestimonialsSection, ContactSection, AdminLogin
  - المكونات الأساسية (Navbar, HeroSection, Footer) تحمل مباشرة
- Success criteria: تحميل أسرع للصفحة الأولى، المكونات تظهر تدريجياً أثناء التمرير

**Font Preloading**
- Functionality: تحميل الخطوط بشكل مسبق لتجنب التأخير
- Purpose: القضاء على Flash of Unstyled Text (FOUT)
- Implementation:
  - preconnect لـ Google Fonts CDN
  - preload للخط Tajawal
  - display=swap في رابط الخط
- Success criteria: الخطوط تظهر فوراً بدون وميض أو تأخير

**HTML Meta Optimization**
- Functionality: تحسين metadata للصفحة
- Purpose: تحسين SEO وسرعة التحميل
- Implementation:
  - إضافة meta description
  - تعيين lang="ar" و dir="rtl"
  - تحسين title للـ SEO
- Success criteria: الموقع محسّن لمحركات البحث والأداء

**Image Format Optimization**
- Functionality: استخدام WebP و lazy loading للصور الخارجية
- Purpose: تقليل حجم الصور وزمن التحميل
- Implementation:
  - استخدام Unsplash مع معاملات التحسين (w=800&h=600&fit=crop)
  - lazy loading="lazy" على جميع صور المعرض
  - LazyImage component مع Intersection Observer
- Success criteria: الصور محمّلة بأحجام مناسبة وبسرعة عالية

## Design Direction
تصميم عصري وأنيق فخم يجمع بين الاحترافية العالية والرقي المطلق، مع لمسات حديثة جريئة ومبتكرة تعكس التميز والطموح والقوة.

تصميم يركز على:
- **البساطة الفاخرة**: واجهة نظيفة وواضحة مع تفاصيل دقيقة راقية
- **الحركة السلسة**: انتقالات وحركات ناعمة تعكس الاحترافية
- **الألوان الجريئة**: لوحة ألوان بنفسجية وزهرية مشرقة ومميزة
- **التباين القوي**: تباين واضح بين العناصر لسهولة القراءة والتركيز
- **التأثيرات البصرية**: ظلال ثلاثية الأبعاد، تدرجات معقدة، وتأثيرات زجاجية
- **الخطوط العريضة**: نصوص قوية وواضحة بأحجام كبيرة
- **المساحات السخية**: استخدام المساحات البيضاء بذكاء لخلق تنفس بصري
- **التفاعلية الذكية**: استجابات فورية وملحوظة لتفاعلات المستخدم

## Color Selection
تصميم عصري وأنيق بألوان بنفسجية وزهرية راقية متطورة تعكس الاحترافية والتميز والفخامة.

- **Primary Color**: بنفسجي غني وعميق `oklch(0.52 0.21 285)` - يمثل الطموح والاحترافية والتميز والقوة
- **Secondary Color**: بنفسجي فاتح أنيق `oklch(0.68 0.18 300)` - يضيف عمق وتدرج راقي وسلس
- **Accent Color**: زهري ديناميكي ومشرق `oklch(0.65 0.24 330)` - للتركيز والحيوية والطاقة، يجذب الانتباه
- **Background**: خلفية فاتحة نقية `oklch(0.99 0.002 280)` - نظيفة وواسعة ومريحة للعين
- **Foreground/Background Pairings**: 
  - Primary (بنفسجي): White text (#FFFFFF) - Ratio 7.2:1 ✓✓
  - Accent (زهري): White text (#FFFFFF) - Ratio 6.1:1 ✓✓
  - Background (فاتح): Dark text - Ratio 17.5:1 ✓✓

## Font Selection
خط عربي عصري وأنيق يعكس الحداثة والاحترافية مع سهولة القراءة.

- **Typographic Hierarchy**:
  - H1 (العنوان الرئيسي): Tajawal Black/72px/tight، للعنوان البطولي والمميز
  - H2 (عناوين الأقسام): Tajawal Black/56px/normal، لعناوين الأقسام الرئيسية
  - H3 (عناوين الكروت): Tajawal Bold/24px/normal، لعناوين الفئات والكروت
  - Body (النص العادي): Tajawal Medium/18px/relaxed، للوصف والمحتوى
  - CTA Buttons (أزرار الإجراء): Tajawal Black/20px/normal، للتأكيد على الأكشن

## Animations
حركات سلسة وراقية جداً تعكس الاحترافية والحداثة والفخامة، مع تركيز على transitions ناعمة وتأثيرات بصرية جذابة ومبهرة.

- **Hover على الأزرار**: تكبير سلس وفخم مع ظل ناعم ثلاثي الأبعاد (scale 1.1-1.25) لإحساس بالتفاعل الراقي والقوة
- **Scroll Reveals**: ظهور العناصر بطريقة سلسة وأنيقة مع fade-in وslide-in عند التمرير، مع تأخير متدرج
- **Hero Section**: خلفية متدرجة متحركة مع overlays متحركة ناعمة وتأثيرات blur فخمة وكرات ضوئية عملاقة
- **Cards Hover**: رفع كبير مع ظل ثلاثي الأبعاد قوي وتدوير أنيق للأيقونات والعناصر الداخلية
- **Gradient Animations**: تأثيرات gradients متحركة بطيئة وناعمة في الخلفيات مع انتقالات سلسة
- **Glass Effects**: تأثيرات زجاجية شفافة محسّنة (glassmorphism) للعناصر التفاعلية مع blur قوي
- **Pulse & Glow**: تأثيرات توهج نابض للعناصر المهمة والشارات والأيقونات
- **Shine Effects**: تأثيرات لمعان متحركة على النصوص والعناوين الرئيسية

- **Components**: 
  - Button (shadcn): للأزرار الرئيسية مع تخصيص أخضر
  - Card (shadcn): للفئات التدريبية وكروت اللاعبين
  - Input + Label + Textarea (shadcn): للفورم
  - Separator (shadcn): للفواصل بين الأقسام
  
- **Customizations**:
  - خلفية gradient للـ Hero مع صورة overlay
  - Grid pattern بأسلوب خطوط الملعب في الخلفية
  - أيقونات من Phosphor Icons بأسلوب رياضي
  
- **States**:
  - Default: أزرار خضراء صلبة مع نص أبيض
  - Hover: تكبير + ظل أقوى
  - Active: تقليص خفيف للضغط
  - Focus: حلقة خضراء مضيئة
  
- **Icon Selection**:
  - Soccer (كرة قدم) للأيقونة الرئيسية
  - Users (مجموعة) للفئات العمرية
  - Trophy (كأس) للنجاح
  - WhatsApp logo للتواصل
  - MapPin للموقع
  
- **Spacing**: 
  - Sections: py-20 للمساحات الكبيرة
  - Cards: gap-8 بين الكروت
  - Content: gap-4 للمحتوى الداخلي
  - Buttons: px-8 py-4 للأزرار الكبيرة
  
- **Mobile**: 
  - Hero: عمودي بالكامل مع نص مركز
  - Grid: من 3 أعمدة إلى عمود واحد
