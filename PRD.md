# أكاديمية المواهب - موقع ويب لأكاديمية كرة قدم احترافية

موقع ويب متكامل لأكاديمية كرة قدم توفر تدريب احترافي لجميع الأعمار مع مدربين معتمدين في معرة مصرين، إدلب.

## الهدف الرئيسي
تقديم منصة رقمية احترافية لأكاديمية المواهب لكرة القدم، تتيح للزوار التعرف على الأكاديمية وخدماتها والتسجيل بسهولة عبر واتساب، مع لوحة تحكم إدارية شاملة.

**صفات تجربة المستخدم:**
1. **احترافية** - تصميم رياضي قوي يعكس جدية الأكاديمية والتزامها بالتطوير
2. **تفاعلية** - تأثيرات حركية سلسة وانتقالات زجاجية تجذب الانتباه
3. **سهلة الاستخدام** - تنقل واضح وأزرار واتساب مباشرة للحجز السريع

**مستوى التعقيد:** 
تطبيق خفيف (Light Application) - موقع تسويقي متعدد الصفحات مع حالة بسيطة ونظام إداري متكامل

## الميزات الأساسية

### 1. صفحة البطل (Hero Section)
- **الوظيفة:** عرض العنوان الرئيسي وجذب الزوار للتسجيل
- **الهدف:** خلق انطباع أول قوي واحترافي
- **المحفز:** تحميل الصفحة الرئيسية
- **التدفق:** تحميل الصفحة → عرض العنوان بتأثيرات → أزرار CTA → نقر الزائر → فتح واتساب
- **معايير النجاح:** وضوح العنوان + سهولة الوصول لزر التسجيل + معدل نقر عالي

### 2. الفئات التدريبية
- **الوظيفة:** عرض فئات التدريب الثلاث (البراعم، الناشئين، الشباب) مع الأسعار
- **الهدف:** مساعدة الأهالي في اختيار الفئة المناسبة
- **المحفز:** تمرير الزائر للأسفل أو نقر "الفئات" من القائمة
- **التدفق:** عرض الفئات → قراءة التفاصيل → نقر "احجز تجربة مجانية" → فتح واتساب برسالة مخصصة
- **معايير النجاح:** وضوح المعلومات + سهولة المقارنة + معدل حجز مرتفع

### 3. ماذا نقدم
- **الوظيفة:** إبراز الخدمات والمميزات الأربع الرئيسية
- **الهدف:** بناء الثقة وإظهار احترافية الأكاديمية
- **المحفز:** تمرير الزائر أو نقر "خدماتنا"
- **التدفق:** عرض البطاقات → قراءة التفاصيل → بناء الثقة → العودة للتسجيل
- **معايير النجاح:** وضوح المميزات + تصميم جذاب + تحفيز للتسجيل

### 4. معرض الصور
- **الوظيفة:** عرض صور الأكاديمية والملاعب والتدريبات
- **الهدف:** إعطاء نظرة واقعية على المنشآت والأجواء
- **المحفز:** تمرير الزائر أو نقر "المعرض"
- **التدفق:** عرض الشبكة → تحميل الصور تدريجياً (lazy loading) → نقر للتكبير
- **معايير النجاح:** سرعة التحميل + جودة الصور + تنوع المحتوى

### 5. نجومنا
- **الوظيفة:** إبراز قصص نجاح اللاعبين السابقين
- **الهدف:** بناء الطموح وإثبات نتائج الأكاديمية
- **المحفز:** تمرير الزائر للقسم
- **التدفق:** عرض بطاقات اللاعبين → قراءة الإنجازات → تحفيز للانضمام
- **معايير النجاح:** وضوح القصص + إلهام الزوار

### 6. تقييمات الأهالي
- **الوظيفة:** عرض آراء أولياء الأمور والتقييمات
- **الهدف:** بناء الثقة من خلال شهادات واقعية
- **المحفز:** تمرير الزائر أو نقر "آراء الأهالي"
- **التدفق:** عرض التقييمات → قراءة الآراء → زيادة الثقة → التسجيل
- **معايير النجاح:** مصداقية التقييمات + تنوع الآراء

### 7. تواصل معنا
- **الوظيفة:** نموذج تسجيل ومعلومات الاتصال والخريطة
- **الهدف:** تسهيل التواصل والتسجيل
- **المحفز:** تمرير أو نقر "تواصل معنا"
- **التدفق:** ملء النموذج → اختيار الفئة والوقت → إرسال عبر واتساب
- **معايير النجاح:** سهولة الملء + وضوح المعلومات + إرسال ناجح

### 8. لوحة الإدارة
- **الوظيفة:** تحكم كامل بمحتوى الموقع دون الحاجة لمطور
- **الهدف:** تمكين المسؤول من التحديث المستمر
- **المحفز:** نقر شعار الأكاديمية + تسجيل دخول
- **التدفق:** نقر الشعار → إدخال البريد والباسورد → فتح اللوحة → تعديل المحتوى → حفظ → تحديث فوري
- **معايير النجاح:** سهولة الوصول + تحديثات فورية + واجهة بسيطة

## التعامل مع الحالات الاستثنائية

- **فشل تحميل الصور:** عرض placeholder احتياطي + إعادة محاولة تلقائية
- **فشل فتح واتساب:** عرض رقم الهاتف للنسخ + رسالة توضيحية
- **إدخال بيانات خاطئة:** validation فورية + رسائل خطأ واضحة بالعربية
- **عدم وجود اتصال بالإنترنت:** رسالة توضيحية + محاولة إعادة الاتصال
- **صلاحيات إدارية:** حماية لوحة الإدارة بتسجيل دخول + بريد وكلمة سر محددة
- **التحميل البطيء:** lazy loading للمكونات + spinner احترافي
- **الشاشات الصغيرة:** تصميم responsive كامل + قائمة منسدلة للموبايل

## التوجه التصميمي

الموقع يجب أن يشعر الزائر بـ:
- **القوة والاحترافية:** من خلال ألوان رياضية واضحة وخطوط عريضة
- **الشفافية والوضوح:** تأثيرات زجاجية glass morphism تعطي عمق وحداثة
- **الحركة والحيوية:** تأثيرات hover وانتقالات سلسة تعكس النشاط الرياضي

## اختيار الألوان

تم اعتماد لوحة ألوان رياضية احترافية مع تدرجات خضراء تمثل الملاعب:

- **اللون الأساسي (Primary):** `oklch(0.50 0.22 160)` - أخضر ملعب قوي يمثل النشاط والطاقة
- **اللون الثانوي (Secondary):** `oklch(0.42 0.18 165)` - أخضر داكن للمصداقية
- **لون التمييز (Accent):** `oklch(0.60 0.24 150)` - أخضر فاتح للتحفيز والحيوية
- **الخلفية:** `oklch(0.98 0.005 180)` - أبيض مائل للأزرق للنظافة
- **النص:** `oklch(0.15 0.01 180)` - رمادي داكن للقراءة المريحة

**التزاوجات والتباين:**
- Primary على أبيض - نسبة 6.2:1 ✓ (ممتاز)
- Accent على أبيض - نسبة 4.8:1 ✓ (جيد)
- أبيض على Primary - نسبة 16.3:1 ✓ (ممتاز)
- النص الداكن على الخلفية - نسبة 15.1:1 ✓ (ممتاز)

## اختيار الخطوط

الموقع يستخدم عائلة **Tajawal** - خط عربي حديث ومقروء:
- متعدد الأوزان (400 للنص العادي، 700-900 للعناوين)
- يدعم العربية بشكل مثالي
- حديث وعصري مع احتفاظه بالاحترافية

**التسلسل الهرمي للخطوط:**
- `H1 (Hero Title):` Tajawal Black / 96px-128px / tight line-height / drop-shadow
- `H2 (Section Title):` Tajawal Black / 56px-72px / tight
- `H3 (Card Title):` Tajawal Black / 30px-40px / medium
- `Body:` Tajawal Bold / 18px-20px / 1.6 line-height
- `Buttons:` Tajawal Black / 20px-24px / 0.8 line-height

## التأثيرات الحركية

التحريك يجب أن يكون **هادف وسلس** لإضافة الحيوية دون إزعاج:

- **تأثيرات الدخول:** fade-in + slide-up للأقسام عند التمرير (0.5-0.7s)
- **تأثيرات Hover:** scale + lift + shadow enhancement (0.3-0.5s cubic-bezier)
- **الخلفية التفاعلية:** pulsing gradients + scaling background image (15-20s)
- **الأزرار:** scale + shadow glow + gradient shift
- **البطاقات:** 3D lift + border glow + shimmer effect
- **الشعار:** rotate + scale + drop-shadow على hover

**القاعدة:** كل حركة يجب أن تخدم تجربة المستخدم وتحفزه على الفعل.

## اختيار المكونات

### مكونات Shadcn المستخدمة:
- **Button:** للـ CTA والتسجيل (مع تخصيص gradients)
- **Card/CardContent:** للفئات والمميزات والنجوم
- **Input/Label/Textarea:** لنموذج التواصل
- **Select:** لاختيار الفئة العمرية والوقت المفضل
- **Tabs:** في لوحة الإدارة لتنظيم الأقسام
- **Dialog:** للـ modals (تم استخدام admin login modal مخصص)
- **Toast (Sonner):** للإشعارات والتأكيدات

### الأيقونات:
**Phosphor Icons** - مكتبة شاملة لأيقونات رياضية واضحة:
- `WhatsappLogo` - لأزرار التواصل
- `Users, Baby, UsersFour` - للفئات العمرية
- `Trophy, Heart, ChartLine, Park` - للمميزات
- `FacebookLogo, InstagramLogo, TiktokLogo` - السوشيال ميديا
- `MapPin, Clock, Calendar` - معلومات التواصل

### حالات المكونات:
- **Default:** ألوان واضحة + shadows متوسطة
- **Hover:** scale 1.05-1.10 + shadow enhancement + border glow
- **Active:** scale 0.98 + shadow reduction
- **Disabled:** opacity 0.5 + cursor not-allowed
- **Focus:** ring color primary + border emphasis

### التباعد:
- Spacing scale: 4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px
- Section padding: 48px (mobile) - 192px (desktop) عمودي
- Container max-width: 1200px
- Gap بين العناصر: 12px-48px حسب السياق

### التجاوب (Mobile):
- **Breakpoint:** 768px
- **Hero:** عمود واحد + حجم خط أصغر + padding أقل
- **الفئات:** شبكة 1 عمود للموبايل، 3 أعمدة للديسكتوب
- **Navbar:** قائمة منسدلة hamburger للموبايل
- **الأزرار:** عرض كامل للموبايل
- **الصور:** object-cover + aspect ratios مناسبة

## البنية التقنية

**Framework:** React 19 + TypeScript + Vite
**Styling:** Tailwind CSS 4 + shadcn/ui components
**Icons:** Phosphor Icons
**State:** React hooks + useKV (Spark persistence)
**Animations:** Framer Motion + CSS animations
**Lazy Loading:** React.lazy + Suspense
**Forms:** React Hook Form (محتمل في المستقبل)
