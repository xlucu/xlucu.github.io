# أكاديمية المواهب - صفحة HTML ثابتة

## 📁 الملفات

تم إنشاء صفحة HTML ثابتة كاملة في الملف:
- `static-site.html`

## 🚀 كيفية الاستخدام

### 1. للاستخدام المحلي
- افتح الملف `static-site.html` مباشرة في أي متصفح
- الصفحة جاهزة للعمل بدون أي إعدادات إضافية

### 2. للنشر على GitHub Pages

#### الطريقة الأولى (الأسرع):
1. أعد تسمية الملف `static-site.html` إلى `index.html`
2. ارفع الملف إلى مستودع GitHub
3. اذهب إلى Settings > Pages
4. اختر branch: main
5. انقر على Save
6. سيكون موقعك متاحاً على: `https://username.github.io/repository-name/`

#### الطريقة الثانية (مع المجلد الحالي):
```bash
# في terminal
git add static-site.html
git commit -m "Add static website"
git push origin main
```

ثم في GitHub:
- Settings > Pages
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)
- Save

بعدها أعد تسمية `static-site.html` إلى `index.html` في GitHub مباشرة
